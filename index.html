Línea del Tiempo Del
Internet

1989

Tim Berners-Lee ideó la World Wide Web.

Hizo una propuesta llamada "Gestión de la Información, una Propuesta" En donde describía cómo organizar documentos científicos definiendo lo que sería la web junto con versiones tempranas de HTTP y HTML.

1990

Nacimiento de IMDb y el Primer Navegador Web.

Col Needham crea un índice de películas en Usenet que luego evoluciona en IMDb. Tim Berners-Lee lanza WorldWideWeb, el primer navegador, aunque su uso es limitado a máquinas NeXT.

1991

Primeros Navegadores y Protocolos.

La Universidad de Minnesota crea Gopher, un protocolo para organizar documentos. Nicola Pellow desarrolla Line Mode Browser, un navegador de solo texto accesible en varios sistemas. Tim Berners-Lee introduce HTTP v0.9, el primer protocolo para la comunicación web.

1992

Desarrollo y Educación Web.

Louise Addis funda WWW Wizards, un grupo clave para educar y desarrollar la web. ViolaWWW, el primer navegador fuera de CERN, introduce tablas, scripts y hojas de estilo. Tim Berners-Lee presenta la URL, estándar para direcciones web.

1993

Avances Iniciales y Popularización.

NCSA lanza Mosaic versión 1.0, revolucionando la navegación web con su interfaz gráfica de usuario y capacidad multimedia integrada. CERN pone la World Wide Web en el dominio público, permitiendo su libre uso y desarrollo.

1994

Directorios, Curiosidades y Lenguajes.

Jerry Yang y David Filo crean "Jerry and David's guide to the World Wide Web," que luego se convertiría en Yahoo!. Guido van Rossum lanza la primera versión estable de Python.

1995

El Despegue del Contenido en Línea.

Se popularizan los diarios en línea como "Carolyn's Diary," precursores de los blogs. Amazon.com y eBay se establecen como los primeros minoristas en línea.

1996

Herramientas y Recursos para el Diseño Web.

Lynda Weinman publica "Designing Web Graphics," un libro fundamental para diseñadores web. Microsoft adquiere FrontPage, un software de autoría web fácil de usar.

1997

Publicidad y Estándares Web.

Se introduce el primer anuncio emergente (pop-up) en Tripod. El W3C publica HTML 3.2, consolidando varias características ya implementadas en los navegadores.

1998

Innovaciones Tecnológicas y Estándares.

El W3C publica la especificación XML 1.0, facilitando el intercambio de datos entre máquinas. Google es incorporado oficialmente, introduciendo su algoritmo PageRank.

1999

Expansión y Evolución de la Web.

Se introducen mejoras en la experiencia del usuario y se avanza en la estandarización, con la introducción de favicons, RSS y las WCAG.

2000

Avances en la Arquitectura y Estándares Web.

Roy Fielding presenta el concepto de Representational State Transfer (REST) en su tesis doctoral, sentando las bases para las APIs modernas.

2001

Expansión y Diversificación de los Blogs.

Boing Boing lanza su sitio web, convirtiéndose en uno de los primeros blogs de enlaces con comentarios. Michel Valdrighi lanza b2, una plataforma de blogs de código abierto.

2002

Nuevos Navegadores y Técnicas de Desarrollo.

Camino, un navegador para Mac, se basa en Firefox y utiliza la API Cocoa. Tantek Çelik introduce el doctype switching y el Box Model Hack.

2003

Mejoras en Navegadores y Técnicas de Desarrollo Web.

Apple lanza Safari, su propio navegador basado en el motor KHTML. Steven Champeon y Nick Finck presentan la técnica de "progressive enhancement".

2004

El Auge de las Redes Sociales y las Aplicaciones Web.

Orkut, una plataforma de redes sociales, gana popularidad en Brasil e India. 37signals lanza Basecamp, una herramienta de gestión de proyectos basada en Ruby on Rails.

2005

Avances en Aplicaciones Web y Navegadores.

Google Maps se lanza, revolucionando la navegación en línea con su interfaz interactiva. YouTube se lanza, transformando la forma en que se comparten y consumen videos en la web.

2006

Herramientas de Desarrollo y Frameworks JavaScript.

Joe Hewitt lanza Firebug, una herramienta de depuración para Firefox. John Resig presenta jQuery, una biblioteca JavaScript que simplifica la manipulación del DOM.

2007

El Surgimiento de la Web Móvil y Herramientas de Desarrollo.

Apple presenta el iPhone, revolucionando la web móvil con su navegador completo y soporte para estándares web.

2008

Herramientas de Colaboración y Recopilación de Información.

Ushahidi se lanza como una plataforma de crowdsourcing para recopilar y mapear informes, especialmente útil en situaciones de crisis.

2009

Preservación y Organización del Contenido Web.

The Archive Team se forma para archivar Geocities antes de su cierre. Natalie Downe introduce los pattern portfolios, una metodología para organizar patrones de marcado y CSS reutilizables.

2010

Plataformas Visuales y Formatos Web.

Pinterest se lanza como una plataforma para recopilar y compartir imágenes y enlaces en tableros digitales.

2011

Iniciativas para la Diversidad y Herramientas de Diseño.

Kimberly Bryant funda Black Girls Code, una organización sin fines de lucro para fomentar el interés de las jóvenes minoritarias en la ingeniería informática.

2012

Protestas y Legislación.

Miles de sitios web, incluyendo Wikipedia y Reddit, realizan una huelga virtual para protestar contra SOPA y PIPA, leyes propuestas que amenazaban la libertad en línea.

2013

Frameworks de Interfaz de Usuario.

Facebook lanza React, un framework para construir interfaces de usuario web basado en componentes que se actualizan automáticamente con los cambios de datos.

2014

Frameworks JavaScript.

Evan You lanza Vue.js, un framework web para construir aplicaciones de una sola página, con una estructura modular que permite a los desarrolladores utilizar solo las partes necesarias.

2015

Aplicaciones Web Progresivas (PWA).

Alex Russell introduce el concepto de Aplicaciones Web Progresivas (PWA), un marco para construir aplicaciones avanzadas en la web utilizando nuevas características del navegador.

2016

Reglamento General de Protección de Datos (GDPR).

La Unión Europea adopta el Reglamento General de Protección de Datos (GDPR), que impone restricciones más estrictas sobre la recopilación de información personal en línea.

2017

Experimento Social en Línea.

Reddit lanza "Place," un experimento del Día de los Inocentes que permite a los usuarios cambiar píxeles en una cuadrícula compartida, fomentando la colaboración a gran escala.
